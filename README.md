## Install Dependencies

Requires Lua and luarocks
### Install `lua` or `lua-jit`
In mac: `brew install lua`

### Install luarocks
In mac: `brew install luarocks`

### Install rocks
`./install_rocks.sh`

## Run
`lua main.lua`
